Este proyecto esta hecho con python y con la libreria pygame recomendado para hacer juegos pequeños
y aprender la programacion de videojuegos, su punto debil es que aunque se puede hacer coliciones no tiene entorno grafico
aunque se pueda trabajar si un entorno grafico, recomiendo que una vez que domines esta libreria uses otra con entorno grafico 
para una experiencia mejor.
Ademas con python y pygame puede ser dificil de subir a una pagina online para que alguien lo pruebe sin necesidad de descargarlo
basado en mi experiencia recomendaria unity que se puede subir tu juego a itch.io, actualmente tengo 2 juegos ahi y se pueden probar por la pagina
sin necesidad descargar.
En casa de tener necesidad de poner pygame o pyturtle en linea:
en mi travesia de querer poner este proyecto en linea y otro, me habia topado con la libreria "Pygbag" es la unica alternativa que habia encontrado
para poner en una pagina mi juego hecho en python que sea comptatible con pygame, si memoria no me falla la libreria usaba  "asyncio" (o uno parecido)
los creadores de la libreria son dos personas de habla ingles y una pequeña comunidad lo que hace que sea dificil 
la pagina para descargar y entender la libreria en cuestion seria esta: https://pypi.org/project/pygbag/
